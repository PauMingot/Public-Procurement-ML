# 🏛️ Data Analysis for Strategic Decision-Making in Public Procurement [WIP]

> **Status:** 🚧 Work in Progress (Expected completion: July 2026)
> **Context:** This project is being developed as my Bachelor's Thesis (TFG) in Mathematics at the University of Alicante.

## 🎯 Project Overview
The objective of this project is to build a machine learning system capable of forecasting the winning bid price of public tenders. By analyzing historical data on competitors, tender specifications, and public procurement trends, this tool aims to provide strategic insights for bidding optimization and competitive advantage.

A key mathematical focus of this thesis is not just the prediction itself, but **explainability**—detailing the mathematical and statistical reasons behind the model's results to support transparent business decision-making.

## 🛠️ Tech Stack & Tools
* **Programming & Modeling:** Python (Scikit-learn)
* **Data Manipulation & Analysis:** Pandas, NumPy
* **Database Management:** SQLite3
* **Data Visualization:** Matplotlib, PowerBI
